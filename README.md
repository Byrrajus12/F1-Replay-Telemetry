# F1-Replay-Telemetry

A Web-based Formula 1 race replay and telemetry platform.

Stack:
- Next.js + PixiJS (Frontend)
- FastAPI + FastF1 (Backend)

## Vision
Create an intuitive browser-based F1 telemetry replay experience.

## Roadmap
- Race replay engine
- Telemetry overlays
- Overtake detection
- Pit stop visualization
- Ghost lap comparison
